#### MDAC（Malaysia Digital Arrival Card）

**マレーシア入国時に必須のオンライン入国カードです。**

| 項目 | 内容 |
|------|------|
| **正式名称** | Malaysia Digital Arrival Card (MDAC) |
| **提出方法** | オンライン（Webサイト） |
| **提出時期** | **到着72時間前〜当日の間のみ**（それ以前は不可） |
| **料金** | 無料 |
| **対象者** | 外国人旅行者全員（シンガポール国籍、外交官、永住者を除く） |

**申請URL：** https://imigresen-online.imi.gov.my/mdac/main

**⚠️ 72時間ルールの詳細：**
- 到着72時間前（3日前）から提出可能
- **それ以前に提出するとエラーで不可**
- 例：6/16到着の場合 → 6/13以降に提出可能
- 提出忘れると入国審査で遅延の原因に

**入力内容：**
- パスポート情報（番号、氏名、生年月日）
- 渡航日、搭乗便名
- 宿泊先住所
- 健康状態・渡航歴

**注意点：**
- ビザとは異なる → ビザ免除でもMDACは必須
- 空港到着後は提出不可 → 事前提出必須
- 偽サイトが多数 → URLに `imi.gov.my` があることを確認
- エラーで再提出可能 → 正しい情報を再入力
